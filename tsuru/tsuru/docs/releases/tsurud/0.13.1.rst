===========================
tsurud 0.13.1 release notes
===========================

tsurud 0.13.1 adds `Docker <https://www.docker.com/>`_ 1.10 support,
fixing some bugs with `app-shell` and `app-run`: `#1365 <https://github.com/tsuru/tsuru/issues/1365>`_.
